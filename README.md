# 拾词

一个可直接部署的 CET-6 词汇学习网页，支持：

- TXT、CSV、Word（DOCX）和 Excel（XLSX）导入
- 自动区分单词与词组
- 按 1、2、4、7、15、30、60 天安排记忆复习
- 随机巩固、词库搜索、删除和 CSV 导出
- 电脑和手机自适应，并支持离线再次打开

## 文件格式

- Word：每行写成“英文, 释义”，或使用两列表格。
- Excel：第一列放英文，第二列放释义；第一行可以是表头。
- 旧版 `.doc` 和 `.xls` 请先在 Office 中另存为 `.docx` 和 `.xlsx`。

## 发布为网址

最简单的方法是使用 Netlify Drop：

1. 打开 `https://app.netlify.com/drop`
2. 将本文件夹整体拖入页面
3. 发布完成后会得到一个 `https://...netlify.app` 网址
4. 电脑和手机都可以直接输入该网址访问

也可以将本目录上传到 GitHub，并在仓库设置中开启 GitHub Pages。

## 数据说明

词库和复习进度保存在当前浏览器中，不会上传到服务器。不同设备的词库不会自动同步，可通过“导出当前词库”在设备之间转移。
